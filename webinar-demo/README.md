# Webinar Demo Pack: OKR Skills

## Goal

Show a simple, practical flow for:

- `$okr-write` (create)
- `$okr-review` (improve)
- `$okr-audit` (stress-test)

Audience should quickly see two things:

- The flow is easy to follow.
- The output quality improves fast.

## 9-Minute Run Of Show

### Minute 0-3: Write

- Open `01-okr-write.md`
- Run `$okr-write`
- Use the short answers from "Minimal human inputs"
- Show final draft block

Presenter line:
"In 3 minutes, Yehor gets a clean first OKR draft with real numbers, owners, and cadence."

### Minute 3-6: Review

- Open `02-okr-review.md`
- Paste the weak draft
- Run `$okr-review`
- Compare before vs after

Presenter line:
"Review mode upgrades fuzzy goals into measurable commitments without rewriting strategy from scratch."

### Minute 6-9: Audit

- Open `03-okr-audit.md`
- Paste overloaded draft with too many Objectives/KRs
- Run `$okr-audit`
- Show severity-ordered issues and revised focused set

Presenter line:
"Audit mode is the pressure test: it finds risk fast and forces focus limits."

## Commands To Use Live

```text
$okr-write
$okr-review
$okr-audit
```

## Reset Between Demos

Run each demo as a standalone example. Do not depend on prior chat memory.  
Each file includes everything needed for that segment.
