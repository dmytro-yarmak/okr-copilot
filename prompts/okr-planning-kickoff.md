# OKR Planning Kickoff Prompt

Use this prompt to start a guided OKR planning session.

## The Prompt

```
I'd like help planning OKRs for [team/individual] for [time period].

Context:
- Company's top priority this quarter: [priority]
- Our team's mission: [mission]
- Biggest challenge we're facing: [challenge]
- Biggest opportunity we see: [opportunity]

Please help me think through this as a co-pilot:
- Ask me one question at a time
- Challenge my thinking but let me decide
- Help me focus - less is more
- Connect my goals to what matters most

Let's start. What should my first consideration be?
```

## Variations

### For a CEO/Leadership Team
```
I'm setting company OKRs for [quarter/year].

Our vision: [vision]
Our mission: [mission]
Current situation: [context]

Help me identify the 1-2 objectives that would have the most strategic impact. One question at a time, please.
```

### For a Team Lead
```
I need to create team OKRs that align with this company objective:
[Company Objective]

My team: [team description]
Our capabilities: [what we're good at]
Our constraints: [limitations]

Help me think through how we can best contribute. What's the first thing I should consider?
```

### For an Individual Contributor
```
I want to set personal OKRs that connect to my team's goals:
[Team OKR]

My role: [role]
My growth areas: [development focus]

Help me find the intersection of team impact and personal growth. Where should I start?
```
